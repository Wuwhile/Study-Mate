# 擎思启学 前端

擎思启学是“基于大模型的个性化资源生成与学习多智能体系统开发”项目的 UniApp 前端。

## 核心功能

- 学情诊断：采集学习目标、基础水平、时间预算和资源偏好。
- 个性化资源库：展示讲义、练习、复习计划、项目任务和考前冲刺材料。
- AI 学习助手：面向课程答疑、错题解析、学习计划和资源生成的对话入口。
- 学习导师预约：预约学习规划导师，沉淀学习需求和预约记录。
- 个人中心：账户管理、设置、帮助、反馈和系统说明。

## 技术栈

- Vue 2
- UniApp
- uView / uni-ui

## 后端配置

接口地址位于 `common/config/env.js`，默认连接部署在服务器上的擎思启学后端服务。
